>Sind programmiersprachliche Konstrukte zur Bildung eines neuen Wertes anhand von vorhandenen Werten

## 3.1 Operanden

>Sind Direktwerte, Zahlen (ganz als dezimal, octal oder hexadezimal oder als ASCII-Zeichen oder gebrochen) oder Zeichen in Apostroph, auch Variablen

### 3.1.1 Konstante Ausdrücke

- enthalten nur konstante Werte als Operanden
- werden vom Compiler ausgerechnet und das Ergebnis in den Code eingetragen
- erzeugen keinen ausführbaren Code zur Laufzeit

### 3.1.2 Variablen

- Operanden können auch Werte aus Variablen sein
- Variablen können Werte aufnehmen, wenn sie links von einem = stehen

### 3.1.3 Arrays

- Operanden können auch Arrays sein
- werden durch einen Index selektiert
- Index beginnt mit 0 und ist ganzzahlig
- in C wird die Gültigkeit eines Index nicht geprüft
- Index als Direktwert, Integervariable oder ganzzahliger Ausdruck


- String wird immer mit NULL terminiert
- Wird automatisch gemacht, allerdings sollte man dafür einen Byte reservieren
- Also wenn man einen string mit `a[34]` definiert, dann darf man nur 33 Zeichen schreiben

```run-c
#include <stdio.h>
#include <stdlib.h>

int main() {
	char a[] = "Karl"; // automatisch festgelegt
	printf("%s\n", a);
	printf("Das Array a hat %zu Bytes.\n", sizeof(a));
	return 0;
}
```

## 3.2 Operatoren

 - bestimmen die auszuführende Operation
   Bsp.: + (Addition)
 - Assoziativität (links-, rechtsassoziativ)
 - Priorität (multiplikativ vor additiv)
 - Anzahl der Operaden (binär, unbinär)

### 3.2.1 Bitverschiebeoperationen

- << bzw. >>
- bei einer Bitverschiebung nach links werden von rechts Nullen eingeschoben
- bei einer Bitverschiebung nach rechts werden von links Nullen eingeschoben, wenn die Zahl positiv ist (die Zahl ist unsigned oder der höchstwertige Bit 0)
- bei einer negativen Zahl (die Zahl ist signed und der höchstwertige Bit 1) werden von links Einsen eingeschoben
